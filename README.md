Liste des outils que MIKO AI utilise : 

- Whisper pour la retranscription vocal (https://openai.com/fr-FR/index/whisper/)

- Face-api.js pour les modèles de visages (disponible dans le dossier "models" puis dans "weights". Pour le web. https://github.comjustadudewhohacks/face-api.js.)

- DeepFace pour la recconaisance facial (pour Python. https://github.com/serengil/deepface.git)
